What This Proves

  • Zombie detection correctly identifies sources stuck in "processing" without a file_path
  • Recent sources (less than the threshold age) are not flagged as zombies
  • Sources with a valid file_path are never flagged as zombies regardless of status
  • The compile route rejects zombie sources with an appropriate error instead of entering an infinite retry loop
  • Valid sources with file paths can still be compiled normally
  • Missing sources return 404, and other users' sources are hidden (multi-tenant isolation)
