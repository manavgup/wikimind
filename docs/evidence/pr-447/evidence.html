PR #447 Evidence: WikiMind MCP Server

Phase 1 All Tests Passing

Phase 1 of issue #447: stdio-mode MCP server exposing 4 read-only tools. Generated 2026-05-08.

1. MCP Tools Registered

The server registers 4 tools accessible via the MCP protocol:

ToolDescriptionRequired Params
wiki_search Search wiki articles by keyword query (string)
wiki_get_article Retrieve full article by ID or slug id_or_slug (string)
wiki_ask Ask a question against the wiki (Q&A agent) question (string)
wiki_list_sources List ingested sources none (all optional)

2. Server Initialization (stdio)

The MCP server starts, initializes the database, and responds to the initialize handshake:

# Request:
{"jsonrpc":"2.0","method":"initialize","params":{
  "protocolVersion":"2025-03-26",
  "capabilities":{},
  "clientInfo":{"name":"test","version":"0.1"}
},"id":1}

# Response:
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "protocolVersion": "2025-03-26",
    "capabilities": {
      "tools": { "listChanged": false },
      "resources": { "subscribe": false, "listChanged": false },
      "prompts": { "listChanged": false }
    },
    "serverInfo": {
      "name": "wikimind",
      "version": "1.27.1"
    },
    "instructions": "WikiMind is a personal LLM-powered knowledge OS. ..."
  }
}

3. tools/list Response

The tools/list method returns all 4 tools with JSON Schema for inputs:

# Request:
{"jsonrpc":"2.0","method":"tools/list","params":{},"id":2}

# Response (abbreviated):
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "tools": [
      {
        "name": "wiki_search",
        "description": "Search wiki articles by keyword...",
        "inputSchema": {
          "properties": {
            "query": { "type": "string" },
            "limit": { "default": 10, "type": "integer" }
          },
          "required": ["query"]
        }
      },
      { "name": "wiki_get_article", ... },
      { "name": "wiki_ask", ... },
      { "name": "wiki_list_sources", ... }
    ]
  }
}

4. CLI Commands

Two new CLI commands under wikimind mcp:

$ wikimind mcp --help
Usage: wikimind mcp [OPTIONS] COMMAND [ARGS]...

  Model Context Protocol server commands.

Commands:
  config  Print Claude Desktop configuration snippet for WikiMind MCP.
  serve   Start the WikiMind MCP server (stdio transport).

$ wikimind mcp config
Add this to your Claude Desktop config (claude_desktop_config.json):

{
  "mcpServers": {
    "wikimind": {
      "command": "python3",
      "args": ["-m", "wikimind.mcp.server"]
    }
  }
}
